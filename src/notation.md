# Mathematical Notation

List of mathematical notation used throughout this repo, for ease of access.

- Chi: χ
- CHI: Χ
- Delta: δ
- DELTA: Δ
- Dot product: •
- Element of: ∈
- Epsilon: ε
- Infinity: ∞
- Lambda: λ
- Nabla: ∇
- Pi: π
- PI: Π
- Set of real numbers: ℝ
- Sigma: σ
- SIGMA: Σ
- Theta: θ
- THETA: Θ
