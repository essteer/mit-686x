# Mathematical Notation

List of mathematical notation used throughout this repo, for ease of access.

- Chi: χ
- CHI: Χ
- Delta: δ
- DELTA: Δ
- Dot product: •
- Element of: ∈
- Elipsis (midline): ⋯
- Elipsis (vertical): ⋮
- Epsilon: ε
- Gamma: γ
- Infinity: ∞
- Integral: ∫
- Lambda: λ
- Nabla: ∇
- Parallel (use for magnitude / norm): ∥
- Perpendicular: ⊥
- Pi: π
- PI: Π
- Plus minus: ±
- Set of real numbers: ℝ
- Sigma: σ
- SIGMA: Σ
- Therefore: ∴
- Theta: θ
- THETA: Θ
